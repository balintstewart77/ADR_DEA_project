# Training and pilot

This folder is for coder-facing training and public pilot documentation used to
define the final exclusions. It must not contain trainer-only answer keys,
personal contact information, reserve IDs, or blinded assignments. Public
training material is expected to be frozen; trainer-only and operational pilot
material remains restricted or hash-committed under a separately approved
procedure. No current coder handout, trainer guide, or final pilot
exclusion/assignment artefact was located.
