# Adjudication and release

This folder is for the frozen adjudication instrument, audit-selection code,
and blank evidence-pack and senior sign-off templates. It must not contain
completed adjudications, reviewer contact details, or final release decisions
before registration. Instruments and code are frozen; blank forms are draft
templates; completed outputs are post-registration. None of the expected
validation-specific artefacts was located. Existing dashboard adjudication tests
are supporting software tests, not the required instrument or audit selector.
