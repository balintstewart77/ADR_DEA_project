# Post-registration outputs

This area is for official draw logs, restricted formal REDCap exports,
preregistration amendments, final results, and adjudication outputs created only
after registration and the relevant protocol step. The empty tracked
directories and `.gitkeep` files are structural placeholders only. They do not
mean that sampling, coding, analysis, adjudication, or release has occurred.

Do not place reserve manifests or personal data in a public path. REDCap
exports remain ignored and require the approved restricted-data workflow.
